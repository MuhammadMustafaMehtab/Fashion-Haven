<h2> Fashion Haven – Static E-commerce Landing Page</h2>

<h3> Overview</h3>
<p>
Fashion Haven is a static e-commerce landing page created using pure HTML5 and CSS3.
This project was built right after completing my core HTML and CSS learning phase to apply layout structuring, section organization, and styling concepts in a real-world design.
</p>

<p>
The goal of this project was to practice building a complete website layout from scratch without using any frameworks or external libraries.
</p>

<h3> Project Purpose</h3>
<p>This landing page was developed to strengthen my understanding of:</p>

<ul>
  <li>Semantic HTML structure</li>
  <li>CSS positioning and layout techniques</li>
  <li>Flexbox alignment</li>
  <li>Section-based page design</li>
  <li>Product card styling</li>
  <li>Navigation bar and footer structuring</li>
</ul>

<h3> Features</h3>
<ul>
  <li>Hero section with promotional banner</li>
  <li>Categories section (Men, Women, Accessories)</li>
  <li>Popular products section with pricing</li>
  <li>Featured collections</li>
  <li>Customer review cards</li>
  <li>Contact section with form layout</li>
  <li>Structured footer with navigation links</li>
</ul>

<h3> Technologies Used</h3>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
</ul>

<p>No frameworks or libraries were used in this project.</p>


<h3> What I Learned</h3>
<ul>
  <li>How to structure complete web pages independently</li>
  <li>Maintaining consistent spacing and alignment</li>
  <li>Designing clean and organized product cards</li>
  <li>Managing multiple sections within a single-page layout</li>
</ul>
