 <h3>Fashion Haven – Static E-commerce Landing Page</h3> 
 <h5>Overview</h5>

<p>Fashion Haven is a static e-commerce landing page created using pure HTML5 and CSS3.
This project was built right after completing my core HTML and CSS learning phase to apply layout structuring, section organisation, and styling concepts in a real-world design.
The goal of this project was to practice building a complete website layout from scratch without using any frameworks or external libraries.</p>

<h5>Project Purpose</h5>

<p>This landing page was developed to strengthen my understanding of:</p>

<ul>
 <li>Semantic HTML structure</li>
 <li>CSS positioning and layout techniques</li>
 <li>Flexbox alignment</li>
 <li>Section-based page design</li>
 <li>Product card styling</li>
 <li>Navigation bar and footer structuring</li>
</ul>

<p>It represents my early frontend development practice and foundational skills.</p>

<h5>Features:</h5>

<ul>
 <li> Hero section with promotional banner</li>
 <li> Categories section (Men, Women, Accessories)</li>
 <li> Popular products section with pricing</li>
 <li> Featured collections</li>
 <li> Customer review cards</li>
 <li> Contact section with form layout</li>
 <li> Structured footer with navigation links</li>
</ul>





 Technologies Used

HTML5

CSS3

No frameworks or libraries were used in this project.


 What I Learned

Through this project, I improved my ability to:

Structure complete web pages independently

Maintain consistent spacing and alignment.

Design clean product cards
Organise multiple sections within a single page.
